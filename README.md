# kitty-collector

화살표 방향에 맞는 고양이 선별 게임


## 게임 방법
중앙에 2종류의 고양이를 올바른 곳으로 보내주세요. 😻😼

올바른 곳에 보내게 되면 점수가 증가하고, 다른 곳에 보내게 되면 게임이 종료돼요.

최대한 많은 고양이들을 올바른 곳으로 보내보세요!
